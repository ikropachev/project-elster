### Project ELSTER
Extract, Load, Save, Transform, Edit, Repair.

ETL-service with multiple datasources. Tech stack: Spring Boot v3, Spring Data JPA, Jakarta, JDK18, H2 database, HSQLDB,
MySQL, PostgreSQL, JUnit5, Swagger v3, Maven.

Work in progress.
